Календарь


Чат-бот Телеграм: Реализован на Python и предназначен для рассылок изменений в мероприятиях и запланированных событий, позволит пользователю указывать виды спорта, о каких он бы хотел получать информацию, время, за сколько дней до мероприятия он бы хотел получать напоминания, ну и просто просмотреть мероприятия, которые его интересуют.

БД: реализована на 1С. Предназначена для хранения информации о мероприятиях, изменениях, видах, местах их проведения и т.п. Также хранится информация о том, какие пользователи на какие рассылки подписаны и их персонализированные настройки уведомлений. В последствии будет выгружена на облако и доступна для просмотра через веб-страницу. Там можно будет просмотреть все мероприятия, сортировать их по виду спорта, датам проведения, месту проведения и так далее. На главной странице диаграмма Ганта, которая отражает все мероприятия по выбранному виду спорта. Через диаграмму можно попасть на подробное описание каждого мероприятия.

Парсинг происходит через Python и заносится в базу данных.
